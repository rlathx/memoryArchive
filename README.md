# 기억 아카이브 (Memory Archive)

## 📌 프로젝트 개요

**기억 아카이브**는 하루의 감정과 생각을 간단히 기록할 수 있는 **감성 일기 웹 서비스**입니다.  
매일 1~2개의 고정 질문(예: "오늘 어떤 감정을 느꼈어?", "지금 나에게 해주고 싶은 말은?")에 대한 답변을 입력하고 저장할 수 있습니다.  
이후 달력 UI를 통해 날짜별 기록을 시각적으로 확인하며, 감정을 색상 또는 이모지로 표현하여 한 달의 감정 흐름을 한눈에 볼 수 있습니다.

---

## 🎯 핵심 개발 방향성

### **"질문 중심 일기"로 기록의 진입 장벽 낮추기**

일반적인 일기 서비스는 사용자가 모든 내용을 직접 작성해야 하기 때문에 꾸준히 기록하는 과정에서 쉽게 지루함을 느낄 수 있습니다. 

**기억 아카이브는 이러한 문제를 다음과 같이 해결합니다:**

1. **고정된 질문 제공**  
   - 매일 동일하거나 추천되는 질문을 통해 "무엇을 쓸까?"라는 고민을 제거
   - 질문에 답하는 형식으로 자연스럽게 하루를 돌아보게 유도

2. **자동 추천 질문 (확장 기능)**  
   - AI 또는 미리 준비된 질문 풀에서 매일 다른 질문을 자동 추천
   - 사용자가 지루함을 느끼지 않도록 다양성 제공

3. **감정 표현의 단순화**  
   - 긴 문장이 아닌, 감정 선택(색상/이모지) + 짧은 답변으로 기록 완료
   - 빠르고 부담 없는 기록 경험 제공

4. **시각적 피드백**  
   - 달력에 감정이 색상으로 표시되어 한 달의 감정 흐름을 한눈에 확인
   - 기록의 성취감과 지속 동기 부여

**→ 평소 일기를 쓰지 않던 사용자에게도 새로운 재미와 동기부여를 제공하며, 꾸준한 기록 습관 형성을 목표로 합니다.**

---

## ✨ 핵심 기능 (MVP)

- 매일 고정된 질문에 대한 답변 입력 및 저장
- 달력 UI를 통한 날짜별 기록 조회 및 수정
- 감정 선택(색상/이모지)과 달력 표시
- 기록의 백업/복원 기능(JSON 내보내기/가져오기)

## 🚀 확장 아이디어

- 고정 질문 변경 및 질문 추천 기능
- 감정 통계 시각화 (Canvas API 또는 Chart.js CDN 활용)
- MBTI 기반 하루 마무리 멘트 제공
- AI 질문 생성기 (매일 다른 질문 자동 추천)
- 익명 계정 기반 간단 로그인 및 기기간 동기화

| 구분 | 사용 기술 | 설명 |
|------|------------|------|

| **시각화 (확장)** | Chart.js (CDN) | 감정 통계 차트 시각화 |

| **디자인 도구** | Figma (선택) | UI 시안 및 화면 설계 |
| **배포** | Render / Railway / Fly.io | 무료 Node.js 호스팅 서비스 |

## 💡 프로젝트 차별점 요약

| 일반 일기 서비스 | 기억 아카이브 |
|------------------|----------------|
| 자유 형식 작성 → 부담감, 지루함 | 질문 중심 답변 → 낮은 진입 장벽 |
| 긴 문장 작성 필요 | 짧은 답변 + 감정 선택으로 빠른 기록 |
| 텍스트 중심 | 시각적 피드백 (달력 색상, 통계) |
| 꾸준한 기록 어려움 | 질문 추천으로 재미와 동기 부여 |

## 🚀 기대 효과

1. **꾸준한 기록 습관 형성**  
   질문에 답하는 형식으로 기록의 부담을 최소화하여 지속 가능한 일기 작성 유도

2. **감정 인식 및 자기 성찰**  
   하루를 돌아보는 질문을 통해 자연스럽게 감정을 인식하고 정리

3. **시각적 성취감**  
   달력과 통계를 통해 한 달간의 감정 변화를 한눈에 확인하며 기록의 가치 체감

4. **접근성 향상**  
   평소 일기를 쓰지 않던 사용자도 쉽게 시작할 수 있는 친근한 서비스

---

> **개발 철학:** 모든 개발은 **HTML, CSS, Node.js (Vanilla JavaScript)** 기반으로 진행하며, 외부 프레임워크 없이 순수 기술 스택으로 구현합니다. "질문 중심 일기"라는 핵심 컨셉을 통해 사용자 경험을 극대화합니다.
